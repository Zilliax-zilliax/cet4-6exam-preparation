# 打字练习 Vue3+Vite

一个基于 Vue3 和 Vite 的打字练习应用，支持用户登录注册功能。

## 功能特点

- 英文单词打字练习
- 实时显示打字速度、准确率和用时
- 用户登录、注册、修改密码功能
- 深色/浅色主题切换
- 响应式设计，适配各种设备

## 安装与运行

### 前端

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build
```

### 后端

```bash
# 进入后端目录
cd 后端登录界面

# 安装依赖
npm install

# 运行后端服务
node app.js
```

后端服务将在 http://localhost:5500 启动

## 后端API接口

- `POST /api/auth/send-captcha` - 发送验证码
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/forgot-password` - 修改密码

## 测试账号

目前后端使用内存存储用户数据，服务重启后数据会丢失。您需要先注册一个账号，然后使用该账号登录。

## 常见问题

1. **验证码无法发送**
   - 确保后端服务已启动
   - 检查网络连接
   - 确保输入了正确格式的邮箱或手机号

2. **登录失败**
   - 确认账号和密码是否正确
   - 确认验证码是否正确
   - 验证码有5分钟有效期

## 注意事项

- 当前版本的后端使用内存存储，重启服务器后数据会丢失
- 验证码会在后端控制台打印，实际项目中应替换为真实的短信/邮件发送服务
