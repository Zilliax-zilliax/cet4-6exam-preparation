# 四六级备考辅助平台

一款基于 Vue + Node.js + MongoDB 开发的英语学习系统，专为四六级备考设计，支持单词练习、收藏管理、AI 单词查询及后台管理功能，项目部署于 Sealos 云平台。

## 项目功能

### 用户端功能
- 用户注册、登录、账号验证
- 四六级单词打字练习，实时错误提醒
- 单词收藏、编辑、删除、搜索
- 收藏单词一键练习
- AI 单词助手，支持单词翻译、音标、短语、例句查询

### 管理端功能
- 管理员登录验证
- 用户信息管理与删除
- 验证码发送记录查看
- 用户数据统计展示

## 技术栈

- 前端：Vue3 + Vite
- 后端：Node.js + Express
- 数据库：MongoDB
- 部署环境：Sealos 云平台

## 系统架构

- 前端页面：提供用户交互界面
- 后端接口：处理业务逻辑与数据操作
- 数据库：存储用户、收藏、练习、管理员等数据
- 云平台：提供稳定运行环境与安全隔离

## 运行环境

- 浏览器：Chrome / Edge 等主流浏览器
- 运行环境：Sealos 云容器环境
- 数据库：MongoDB

## 项目特点

- 界面简洁，操作便捷
- 功能完整，满足四六级学习需求
- 前后端分离，结构清晰
- 基于云平台部署，稳定安全
- 支持个性化单词学习与管理

## 使用说明

1. 注册账号并登录系统
2. 进入单词练习页面进行打字训练
3. 收藏重点单词并在个人收藏页管理
4. 使用 AI 单词助手查询单词信息
5. 管理员可登录后台进行用户与记录管理

## 项目说明

本项目为毕业设计/课程设计作品，功能完整、运行稳定，可直接部署使用，也可作为学习前后端分离项目的参考案例。
