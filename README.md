# README.md
## 项目名称<br>四六级备考辅助平台

## 项目描述<br>本项目是一款面向大学生的英语学习辅助工具，专注于帮助用户提升英语单词掌握能力与拼写准确性，<br>核心围绕四六级单词学习展开，提供打字练习、单词收藏、AI查询等功能，同时支持后台管理与权限控制，<br>整体采用容器化部署，操作便捷、功能完善，可满足用户日常备考与学习需求。

## 技术栈<br>- 前端：Vue、Vite<br>- 后端：Node.js、Express框架<br>- 数据库：MongoDB<br>- 部署环境：Sealos云平台<br>- 核心依赖：无额外复杂插件，确保运行稳定

## 核心功能<br>### 1. 用户端功能<br>- 注册/登录：完成账号注册与登录验证，支持验证码安全校验<br>- 单词练习：提供四六级单词打字练习，实时反馈输入正确性<br>- 单词收藏：可收藏重点单词，支持备注与管理，构建专属单词本<br>- AI查询：通过AI助手查询单词释义、短语、例句，辅助单词记忆<br>### 2. 管理端功能<br>- 管理员登录：专属账号登录，实现权限隔离<br>- 用户管理：查看、删除用户信息，维护平台秩序<br>- 验证码管理：查看验证码发送记录，保障账号安全

## 运行步骤<br>### 本地运行<br>1. 克隆项目至本地，分别进入前端、后端目录<br>2. 前端：执行npm install安装依赖，npm run dev启动前端服务<br>3. 后端：配置MongoDB连接信息，执行node app.js启动后端服务<br>4. 浏览器访问本地端口，即可使用系统<br>### 云端部署（Sealos）<br>1. 登录Sealos云平台，创建容器实例<br>2. 分别部署前端、后端服务及MongoDB数据库<br>3. 配置网络访问策略，开放对应端口<br>4. 验证服务正常运行后，即可通过公网访问使用

## 注意事项<br>1. 运行前需确保MongoDB服务正常启动，配置正确的数据库连接地址<br>2. 前端依赖需完整安装，避免出现界面加载异常<br>3. 管理员账号仅用于后台运维，禁止随意修改用户数据<br>4. 确保Sealos容器环境稳定，避免服务中断

## 后续优化方向<br>1. 拓展听力训练、阅读专项练习等功能模块<br>2. 完善学习数据统计，生成个性化学习报告<br>3. 优化界面交互，提升用户使用体验<br>4. 增加单词分类与难度分级，适配不同学习需求
